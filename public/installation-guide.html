Blood Donation System - Installation Guide

This guide will help you install the Blood Donation System on your cPanel hosting.

Step 1: Server Requirements

Before you begin, ensure your server meets the following requirements:

  • PHP 7.4 or higher
  • PostgreSQL 12 or higher
  • Apache or Nginx web server
  • SSL certificate (recommended for security)

Step 2: Upload Files

Upload the application files to your cPanel hosting:

  1. Login to your cPanel account
  2. Navigate to the File Manager
  3. Go to the public_html directory (or subdirectory where you want to install)
  4. Upload the zipped application package
  5. Extract the files

Step 3: Create Database

Create a PostgreSQL database for the application:

  1. In cPanel, navigate to "PostgreSQL Databases"
  2. Create a new database
  3. Create a database user and assign it to the database with all privileges
  4. Note down the database name, username, and password

Step 4: Configure Installation

Navigate to the installation wizard by visiting your domain:

https://your-domain.com/install

Follow the installation wizard which will guide you through:

  • Database configuration
  • Admin user creation
  • Initial system settings

Note: The installation script will create necessary database tables and initial configuration.

Step 5: Finalize Installation

After completing the wizard:

  1. Set proper file permissions (typically 755 for directories and 644 for files)
  2. Remove or rename the install directory for security
  3. Configure cron jobs if needed (for scheduling tasks)

Manual Installation (Advanced)

If you prefer a manual installation, follow these steps:

  1. Copy the config.php.example file to config.php
  2. Edit the configuration file with your database credentials
  3. Import the SQL schema from database_setup.sql to your PostgreSQL database
  4. Create an admin user directly in the database
-- SQL to create admin user manually
INSERT INTO users (email, password_hash, name, role)
VALUES ('admin@example.com', 'YOUR_HASHED_PASSWORD', 'Admin', 'admin');

Troubleshooting

  • If you encounter database connection issues, verify your database credentials
  • For permission errors, check file ownership and permissions
  • Check PHP error logs in cPanel for detailed error messages
  • Ensure your server meets all the requirements listed above
